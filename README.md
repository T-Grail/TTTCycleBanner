# TTCycleBanner
支持图片、文字的轮播图，分页小圆点可定制。支持非全屏的collectionView分页滚动。

### feature   

- [x] 无限滚动
- [x] 自动滚动
- [x] 页面消失自动暂停、页面展示自动开启
- [x] 图片、文字、富文本文字、占位图，可自定义cell
- [x] TTPagingCollectionView支持自定义宽度的paging
- [x] TTPagingCollectionView单个cell支持自适应宽度
